# 串口与定时器控制流水灯

## 功能描述

通过串口发送指令，控制 PA1 - PA5 五个LED灯的流水行为

## 具体指令

| 指令        | 效果                          |
| ----------- | ----------------------------- |
| `start`     | 开始流水灯，每 200ms 依次点亮 |
| `stop`      | 停止流水灯，所有 LED 熄灭     |
| `speed:100` | 修改流水间隔为 100ms          |
| `speed:500` | 修改流水间隔为 500ms          |

## 预期结果

| 发送指令            | 效果                              |
| ------------------- | --------------------------------- |
| `start`             | PA1→PA2→PA3→PA4→PA5 每 200ms 流水 |
| `stop`              | 全灭，停止流水                    |
| `speed:100`         | 加速到 100ms                      |
| `speed:500`         | 减速到 500ms                      |
| `speed:1000`        | 减速到 1000ms                     |
| 停止时发`speed:xxx` | 无效，保持熄灭                    |
