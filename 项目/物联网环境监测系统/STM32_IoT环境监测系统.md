# 基于STM32+ESP8266的IoT环境监测系统

> 基于STM32F103C8T6 + ESP8266的IoT环境监测系统，实现I2C传感器驱动、UART AT指令控制WiFi模块、MQTT协议云端数据上报、FreeRTOS多任务调度及低功耗设计。

## 硬件清单

| 模块         | 型号                    | 数量 | 用途               |      |
| ------------ | ----------------------- | ---- | ------------------ | ---- |
| 主控         | STM32F103C8T6最小系统板 | 1    | 系统主控           |      |
| 温湿度传感器 | SHT30                   | 1    | I2C采集温湿度      |      |
| WiFi模块     | ESP-01S（ESP8266）      | 1    | MQTT无线上报       |      |
| OLED显示屏   | 0.96寸 I2C SSD1306      | 1    | 本地显示数据       |      |
| USB转TTL     | CH340                   | 1    | 调试ESP8266 AT指令 |      |

## 系统流程

```
SHT30
  │ I2C（每5秒）
  ▼
STM32F103C8T6
  ├─ 解析数据 + CRC校验
  ├─ OLED显示温湿度
  │
  │ UART AT指令
  ▼
ESP8266
  │ MQTT
  ▼
云端平台
```

## 四、通信协议

### STM32 ↔ SHT30（I2C）

- 地址：0x44（ADDR接GND）
- 采集命令：0x2C06（单次高精度）
- 返回6字节：温度2字节 + CRC1字节 + 湿度2字节 + CRC1字节

### STM32 ↔ ESP8266（UART）

- 波特率：115200
- 通过AT指令控制：连接WiFi、建立MQTT连接、发布数据

### MQTT数据格式

```json
{
  "temperature": 25.6,
  "humidity": 60.2,
  "timestamp": 1700000000
}
```
