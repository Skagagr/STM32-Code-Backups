# STM32-Code-Backups
基于STM32F103C8T6最小系统板，使用HAL库与CubeMX制作的各种驱动代码，此仓库用于备份这些代码，以供日后使用时方便

AI 辅助编写的驱动代码

只保留 main 与手动编写的如 uart、timer等代码，软件自动生成的库代码不会保留



## 环境配置

软件：

​	VS Code

插件：

​	翻译(英汉词典)

​	C/C++ Extension Pack	及其扩展包

​	Chinese (Simplified) (简体中文) Language Pack for Visual Studio Code

​	CMake

​	Doxygen Documentation Generator

​	Serial Monitor

​	STM32CubeIDE for Visual Studio Code	及其扩展包

​	Tiny Light

字体：

​	MapleMono-NF-CN-Medium.ttf
