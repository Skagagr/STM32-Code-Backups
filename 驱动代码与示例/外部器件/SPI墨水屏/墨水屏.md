## 引脚配置

| 驱动板引脚 | STM32引脚 | 说明                  |
| ---------- | --------- | --------------------- |
| GND        | GND       | —                     |
| 3.3V       | 3.3V      | —                     |
| SCK        | PA5       | SPI1_SCK，固定不能动  |
| SDA        | PA7       | SPI1_MOSI，固定不能动 |
| RST        | PB12      | 连续GPIO              |
| DC         | PB13      | 连续GPIO              |
| CS1        | PB14      | 连续GPIO              |
| BUSY       | PB15      | 连续GPIO              |
