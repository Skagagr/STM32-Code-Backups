# ADC模数转换

## 热敏电阻模块

以下是用到的电路结构：

```
VCC
 │
R1 (10KΩ 上拉电阻)
 │
 ├── C1(104滤波) ────→ AC 引脚（模拟输出）→ PA0
 │
热敏电阻 (NTC)
 │
GND
```



## 需要更改的代码

因热敏电阻 + 10K 上拉电阻，阻抗相对较高，所以需要 239.5 个周期让采样电容充分充电，才能得到准确的电压值

自行更改 MX_ADC1_Init()

```
sConfig.SamplingTime = ADC_SAMPLETIME_239CYCLES_5;		// 采样周期
```



## 每个热敏电阻的 B 值都是不一样的

我不知道我用的热敏电阻的型号，所以当作最常见的一种

下面的 B 值代码在 adc.h 中可以看到

```
/** @brief 热敏电阻 B 值，常见 NTC 10K 约为 3950 */
#define NTC_B_VALUE     3950.0f
```
